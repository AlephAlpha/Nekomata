## 2. Is it Prime? (CGCC #57617)

```

```

A single built-in function. `Q` (`\isPrime`) checks if the input is prime. For non-deterministic output modes:

- `all` / `first` / `last`: outputs the input value unchanged if it is prime; outputs nothing if not prime
- `exists` (`-e`): outputs `True` if prime, `False` otherwise

## 5. How Even? / 2-Adic Valuation (CGCC #71833)

```

```

With input `n`:
- `À°` (`\lengthWhile`) applies `¬Ω` (`\half`) repeatedly until it fails

Because `48 / 2 = 24`, `/2 = 12`, `/2 = 6`, `/2 = 3` (odd), that's 4 steps.

## 6. Fibonacci (Infinite Generator) (CGCC #85)

```
1:·∂¶{$·µâ+
```

Step by step:

1. `1` pushes `1`
2. `:` (`\dup`) duplicates, giving stack `1 1`
3. `·∂¶{...}` (`\iterate`) applies the block 0 or more times non-deterministically, returning every intermediate result
4. Inside the block:
   - `$` (`\swap`) swaps the top two values
   - `·µâ` (`\dupDip`) applies `+` then pushes the original top back

The iterate function returns every intermediate stack state. Starting from `1 1`, each iteration advances to the next Fibonacci pair:
- **0 iterations**: stack `1 1`, output `1`
- **1 iteration**: `1 1` ‚Üí swap (`1 1`) ‚Üí dup-dip add: `1+1=2`, push original top `1` ‚Üí stack `2 1`, output `1`
- **2 iterations**: `2 1` ‚Üí swap (`1 2`) ‚Üí dup-dip add: `1+2=3`, push original top `2` ‚Üí stack `3 2`, output `2`
- **3 iterations**: `3 2` ‚Üí swap (`2 3`) ‚Üí dup-dip add: `2+3=5`, push original top `3` ‚Üí stack `5 3`, output `3`

The results are `1, 1, 2, 3, 5, 8, 13, ...` ‚Äî the Fibonacci numbers!

**Bytes:** 7

## 9. Collatz Steps (CGCC #12177)

```
À°{∆µ·µâ¬Ω3*‚ÜíI
```

Counts how many steps to reach 1 in the Collatz conjecture.

1. `À°{...}` (`\lengthWhile`) counts applications until failure
2. `∆µ` (`\isBig`) checks if absolute value > 1 (i.e., not 1)
3. `·µâ` (`\dupDip`) applies a function then restores top
4. `¬Ω` halves (for even numbers)
5. `3*‚Üí` computes `3n+1`
6. `I` (`\if`) chooses the first succeeding branch: even ‚Üí halve, odd ‚Üí `3n+1`

```
>>> \Input 10
>>> À°{∆µ·µâ¬Ω3*‚ÜíI
6
```

(10 takes 6 steps to reach 1: 10‚Üí5‚Üí16‚Üí8‚Üí4‚Üí2‚Üí1)

**Bytes:** 9
